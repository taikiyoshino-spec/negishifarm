# 根岸FARM ホームページ

青森県のブルーベリー農園「根岸FARM」の公式サイトです。  
HTML・CSS・JavaScript のみで構成され、スマホ優先のレスポンシブデザインです。

## 構成

- **index.html** … メインのHTML（商品一覧・連絡先・アクセス・新着情報・ギャラリー）
- **styles.css** … モバイルファーストのスタイル
- **script.js** … メニュー開閉・スムーススクロール・ギャラリーライトボックス

## カスタマイズ方法

### 連絡先（index.html の `#contact` 付近）

- **お名前** … `根岸 〇〇（ねぎし 〇〇）` を実際の名前に変更
- **電話番号** … `000-0000-0000` を実際の番号に変更（`<a href="tel:...">` も合わせて変更）
- **メール** … `info@example.com` を実際のアドレスに変更

### アクセス

- 地図の座標は指定の Google マップ URL（40.392525, 141.4846402）で設定済みです。
- 住所表記（〒031-xxxx 青森県八戸市付近）は実際の住所に合わせて変更してください。

### Googleカレンダー

- **予定の表示**: 現在は日本の祝日カレンダーをサンプル表示しています。
- **ご自身のカレンダーに差し替え**: [Googleカレンダー](https://calendar.google.com) → 設定（歯車）→ 左メニューで対象のカレンダーを選択 → 「カレンダーの統合」→ 「埋め込みコード」をコピー → `index.html` の `calendar-iframe` の `src` を差し替えてください。
- **注意**: カレンダーを「一般公開」に設定する必要があります。

### 新着情報（Twitter / X）

- **リンク** … `id="twitterLink"` の `<a>` の `href` を公式 X（Twitter）のURLに変更
- **タイムライン埋め込み** … X の「タイムラインを埋め込む」で取得したコードを、`id="twitterEmbed"` の div 内に貼り付けてください。

### ギャラリー

- 現在は色のプレースホルダーです。実際の写真を使う場合は、各 `.gallery-thumb` の中に `<img src="画像URL" alt="説明">` を追加し、必要に応じて `data-full="大サイズ画像URL"` を付けるとライトボックスで拡大表示できます。

## ローカルで確認

`index.html` をブラウザで開くか、簡易サーバーで表示してください。

```bash
# 例: Python 3
python -m http.server 8080
# ブラウザで http://localhost:8080 を開く
```

## 対象デバイス

- スマートフォン（優先）
- タブレット・PC（600px / 960px でレイアウトが切り替わります）
